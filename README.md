# GlassReader

A lightweight native macOS reader designed for local reading.

GlassReader is a free-to-use native macOS application for reading comics, PDF documents, images, and compressed archives. GlassReader is actively developed and new features are added regularly.
<img width="1360" height="864" alt="截屏2026-07-06 16 01 31" src="https://github.com/user-attachments/assets/ff0840f0-ba89-47fa-9784-d9984b2475db" />

---

## Features

- 📁 Open local folders
- 📦 Read ZIP / CBZ / RAR / 7Z archives directly
- 📄 PDF document support
- 🖼️ JPG / PNG / WebP image support
- 📖 Single-page & Double-page reading
- ↔️ Left-to-right & Right-to-left reading modes
- 🌍 Multi-language interface
  - English、中文、日本語、한국어、Русский、Français、Deutsch、Español
- 🕘 Reading history
- ▶️ Auto Play (Slide Show)
- 🖥️ Native full-screen reading experience
- 🍎 Built exclusively for macOS

---

## Supported Formats

- Folder、ZIP、CBZ、RAR、7Z、PDF、JPG、PNG、WebP

# GlassReader阅读器

一款专注于本地阅读体验的轻量级原生 macOS 阅读器。

GlassReader 是一款免费使用的 macOS 阅读器，可用于阅读漫画、PDF 文档、图片以及各种压缩包文件，拥有简洁的界面、流畅的阅读体验以及完全离线的本地阅读方式。GlassReader 正在持续开发中，未来将持续增加更多阅读格式和功能。

## 功能特点

- 📁 支持打开本地文件夹
- 📦 支持 ZIP / CBZ / RAR / 7Z 压缩包直接阅读
- 📄 支持 PDF 文档
- 🖼️ 支持 JPG / PNG / WebP 图片
- 📖 支持单页 / 双页阅读模式
- ↔️ 支持左右阅读方向切换
- 🌍 支持多语言界面

- 🕘 阅读历史记录
- ▶️ 自动播放（幻灯片）
- 🖥️ 原生 macOS 全屏阅读体验
- 🍎 专为 macOS 打造
