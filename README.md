# GlassReader

> A lightweight native macOS reader designed for local reading.

## Features

- 📖 Read ZIP / RAR / 7Z archives directly
- 📄 PDF support
- 🖼️ Image browsing
- 📚 Single & Double Page Mode
- 🔄 Reading Direction Switching
- 🕒 Reading History
- ▶️ Auto Play
- 🖥️ Full Screen Experience
