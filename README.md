# GlassReader

A lightweight native macOS reader designed for local reading.

GlassReader is a free native macOS application that supports comics, documents and images in one place. It focuses on a clean interface, smooth reading experience and offline privacy.

---

## Features

- 📚 Read ZIP / RAR / 7Z comic archives directly
- 📄 PDF document support
- 🖼️ Image browsing
- 📖 Single-page & Double-page reading
- ↔️ Left-to-right / Right-to-left reading modes
- 🌍 Multi-language interface
  - English
  - 中文
  - 日本語
  - 한국어
  - Русский
  - Français
  - Deutsch
  - Español
- 🕘 Reading history
- ▶️ Auto Play (Slide Show)
- 🖥️ Native full-screen reading experience
- 🍎 Built exclusively for macOS

---

## Supported Formats

- ZIP
- RAR
- 7Z
- PDF
- JPG
- JPEG
- PNG
- WEBP
- GIF
- BMP
- TIFF

# GlassReader 玻璃阅读器

GlassReader 是一款轻量级原生 macOS 阅读器，专注于本地阅读体验。

它支持漫画压缩包、PDF 文档以及各种图片格式，界面简洁，运行流畅，无需联网即可完成阅读。

---

- 📚 支持 ZIP / RAR / 7Z 漫画压缩包直接阅读

- 📖 支持单页 / 双页阅读模式
- ↔️ 支持左右阅读方向切换
- 🌍 支持多语言界面

  - 中文
  - 日本語
  - 한국어
